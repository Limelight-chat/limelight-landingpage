# Getting Started with Limelight

Welcome to Limelight! This guide will help you get up and running with our AI-powered search engine.

## Creating an Account

To start using Limelight, you'll need to create an account. Visit our [signup page](/signup) and enter your details.

## First Steps

Once you've logged in, you can start by:

1.  Connecting your data sources.
2.  Configuring your search settings.
3.  Inviting team members.

## Need Help?

If you run into any issues, check out our [Helpbook](/docs/helpbook) or contact support.
